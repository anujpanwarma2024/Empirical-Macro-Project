# Empirical Macroeconomics — US-India Financial Linkages

Examined transmission of U.S. monetary shocks to Indian financial markets using weekly data on treasury yields (3M/2Y/10Y), USD/INR, NIFTY-50, and S&P 500. 

Methodology covers HP-filter trend-cycle decomposition, AR(1) mean-reversion modelling, GARCH(1,1) conditional volatility estimation, and VAR/SVAR impulse response analysis across both economies.

Data sourced from FRED, NSE, and Investing.com. Built in MATLAB.

Key questions: Do U.S. rate changes spill over into Indian yields and equity returns? How quickly, and through which channel?
